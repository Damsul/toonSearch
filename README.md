# toonSearch

---
## 웹툰 검색 서비스

---

### 프로젝트 소개
>언젠가 볼려고 기억해뒀던 웹툰을 검색하면
> 어디 플랫폼에서 볼 수 있는지 알려주는 서비스

### 프로젝트 기능
>회원
> ---
>* 회원 기능(회원가입, 로그인, 로그아웃, 탈퇴)
>* 검색 기능
>* 정렬 기능(제목순, 평점순)
>* 댓글 기능(댓글 등록, 수정, 삭제)
>* 별점 기능
>* 게시판 기능
>
>관리자
> ---
>* 회원 정지/탈퇴 기능
>* 카테고리 구분 기능
>* 게시판 관리 기능
---
### 기술 스택
* Spring Boot
* Java
* Gradle
* MariaDB
